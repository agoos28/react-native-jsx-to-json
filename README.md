# Requirements
- nodejs
- yarn

# Instalation
In the project folder
```sh
yarn
```
# Start
```sh
yarn start
```
open browser -> http://localhost:2000

# Using Docker
In the project folder

```sh
docker build -t react-native-jsx-to-json
docker run -p 2000:80 react-native-jsx-to-json
```
open browser -> http://localhost
